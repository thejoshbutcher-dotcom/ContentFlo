# UI Guidelines

## Border Radius

-   Cards: 20px
-   Buttons: 14px
-   Inputs: 12px
-   Badges: 999px

## Shadow

``` css
box-shadow: 0 10px 30px rgba(0,0,0,.18);
```

## Animation

-   Hover: 150ms
-   Cards: 200ms
-   Modals: 250ms

Use: - Fade - Scale - Slide

Never bounce.
